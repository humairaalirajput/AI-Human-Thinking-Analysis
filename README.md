# AI and Human Thinking Analysis

##  Project Overview
This project analyzes how Artificial Intelligence (AI) affects human thinking, decision making, and confidence levels.

##  Objectives
- Analyze AI impact on human decision making  
- Study confidence changes after AI advice  
- Examine human reliance on AI  
- Evaluate critical thinking behavior  

## Dataset
Human-AI Interaction Dataset (CSV format)  
Includes:
- Participant ID  
- Initial Response  
- AI Advice  
- Final Response  
- Confidence Level  

## Tools Used
- Orange Data Mining  
- Machine Learning Models:
  - Random Forest  
  - Naïve Bayes  
  - Logistic Regression  
  - kNN  

##  Results
- Random Forest achieved highest accuracy (99.9%)  
- AI improves decision making  
- Overuse of AI reduces critical thinking  
- Confidence increases after AI suggestions  

## Conclusion
AI is helpful but should be used carefully. Overdependence can reduce independent thinking.
